🌐 ScrapeThisSite Web Scraper

A simple Python web scraper that extracts country information from **ScrapeThisSite** using **Requests** and **BeautifulSoup**, then saves the data to a CSV file with **Pandas**.

Features

* Scrapes country details
* Extracts Country, Capital, Population, and Area
* Saves data to `output.csv`

Technologies

* Python
* Requests
* BeautifulSoup4
* Pandas

How to Run


pip install -r requirements.txt
python scraper.py
